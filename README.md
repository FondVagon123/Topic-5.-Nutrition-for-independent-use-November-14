1. МЕТОДОЛОГІЯ ТА ВИКОНАНІ ОБЧИСЛЕННЯ
1.1. Обчислювальні формули

Скрипт виконує основні розрахунки для формування фінансових показників:

Прибуток/од. в грн = Прибуток/од. (у.о.) × 40.5

Квартал — визначено автоматично за датою початку проекту

Заг. прибуток грн (тис.) = Кількість продано (тис.) × Прибуток/од. в грн
